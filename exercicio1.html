Motivacao

HTML é a linguagem utilizada para a construção de sites. É um acrônimo inglês Hypertext Markup Language ou em português Linguagem de Marcação de Hipertexto. Em resumo é a linguagem que você utiliza para demarcar o seu texto, onde você informa ao navegador web qual parte do seu documento se refere um determinado trecho. Ex: < h1>Título do documento< /h1>. O texto anterior refere-se a um título de um documento qualquer e está delimitado entre as tags de abertura e fechamento < h1>.
